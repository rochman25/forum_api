# forum_api
forum api build with node js and hapi.


#how to instal
1. `clone`
2. `cd forum_api`
3. `npm install`
4. `cp .env.example .env`
5. Setting env
6. Setting config/database/test.json set with test environment
7. `npm run migrate up`
8. `npm run migrate:test up`
9. `npm run start:dev`
10. `npm run test`
